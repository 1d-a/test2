# 三层各咋选

# 1.功能型定格：剧情约束→之前的台词与动作

根据之前的动作和台词，找到比较符合的功能型定格类型

- <u>**如果有不相上下的几个选项，那就按情绪滤镜排序来**</u>

  即优先选一本正经>倔强>温柔>小心翼翼......相关的功能型定格类型

# 2.情绪滤镜：综合反差强度、讨喜可爱度，排序——

1. ​**一本正经滤镜**：坐得笔直、动作规矩、像值班
2. ​**倔强滤镜**：不松爪、不挪窝、死守规则
3. **温柔滤镜**：动作很轻、像怕弄疼你
4. ​**小心翼翼滤镜**：只推一厘米、只拍两下、只露半张脸
5. ​**困惑滤镜**：歪头、尾巴像问号、眼神在对焦
6. **得意/自信滤镜**：抬下巴、胸口挺、尾巴竖（偏“我很有用”）
7. ​**委屈滤镜**：尾巴绕脚、耳朵略垂、眼睛湿漉漉
8. **护短滤镜**：它认为“你也不许说你自己”，立刻挡回去

[^1]: # 猫猫型微信号-选部位

    # 筛选1：主收束 还是 主可爱

    **对别人的行为＝强收束；只可爱自己＝强表情**

    - 特殊：**占位一躺，仍算强收束。**

    即——

    对别人的行为（拦、按、挡、推回去、让他别做/必须做），就用下述强收束的部位；

    自己的状态（懵、加载、歪头、对焦、委屈贴贴），就用下述强可爱的部位；

    特别情况--如果是「“占位/压住”形成屏障」（趴在门槛，坐在桌子上），也用强收束

    ###### 强收束

    排序：

    1. ​**全身姿态 + 位置关系**（趴在门槛、挡在嘴前、坐成路障）
    2. ​**爪子细节**（爪尖收着、按住不松、揉一揉）

    ###### 强表情/强可爱

    不分先后：

    1. ​**头部轮廓**（歪头、缩脖、下巴抬起）
    2. **眼睛**（一眨不眨、圆睁、半眯）
    3. ​**耳朵**（竖起、向后贴一点、微微转向）
    4. ​**尾巴**（绷直、绕住、尖尖动一下）
    5. **胡须/毛发/鼻尖**（很像“精细镜头”，容易不够稳定或占字）......很少用

    # 筛选2：只写一个信号位+没被功能动作占用

    - 尾巴 / 耳朵 / 眼睛 / 爪子 / 身形轮廓（面包趴、拱起、缩成团）——只写一个信号位
    - **<u>不能被功能动作占用</u>**

      - 你功能已经写“按住嘴/抱住手腕”（爪子很忙）  
        → 微信号就选 ​**眼睛/耳朵/尾巴**，别再重复“爪子怎么怎么”
      - <u>你功能是“占位趴着”（身形已明显）→ 微信号选 </u>​**<u>尾巴/耳朵</u>**​<u>点一下就够</u>


[^2]: # 猫猫型微信号-选部位动作

    # 筛选1：原则——【静态动作】+【**微幅度/完全静止】**

    ###### 静态动作

    最后一行是截图式定格画面，那所写的动作，就必须是【一幅图能展现出来的】动作。

    包含：

    1. 静态动作

        还、仍、一直、就、死死、牢牢、不动、没松

        例：还按着 / 一眨不眨 / 耳朵竖着不动 / 尾巴绷成一条线 / 趴在那儿不挪
    2. 图画能表达的动态动作

        很多画都有动感，通过静态截图、展现出动态动感——这类动态动作，也可以写

        水被它推到了嘴边（类比画，就是一只爪爪和一杯水）

    ###### **微幅度/完全静止**

    大动作会像推进，小动作更像截图。

    所有优先微动作/微幅度 或 完全静止

    # 筛选2：对齐之前的两条

    - ​**对齐功能**：这个部位在末行要表达的“唯一信息”是什么？（拦住/守住/确认/委屈/得意…）
    - ​**对齐滤镜**：同一信息用什么气质呈现？（一本正经/倔强/困惑/谨慎/委屈/得意）

    还有**压字数**：末行通常 **8–14字** 最稳；过长会像叙述，不像截图

    # 筛选3：高胜率动作的不断积累

    ###### 强收束之「**全身姿态 + 位置关系」**

    - ​**占位**：趴/蹲/坐在关键位置
    - ​**不挪**：不动/不让/堵着
    - ​**挡住**：挡在前面/夹在中间

    ###### 强收束之**爪子细节**

    - ​**不松**（状态化：还按着/没松爪）
    - ​**压住**（证物/危险物被控制）
    - ​**爪尖收着**（谨慎/小心翼翼）
    - ​**轻轻搭着**（安抚/确认）

    ###### 强表情之**头部轮廓**

    - ​**歪头**（困惑/确认）
    - ​**缩脖**（谨慎/怂但认真）
    - ​**抬下巴**（得意/一本正经）
    - ​**定住**（加载失败）

    ###### 强表情之**眼睛**

    - ​**一眨不眨**（一本正经/检查/执法）
    - ​**圆睁**（震惊/当真）
    - ​**半眯**（怀疑/审视/嫌弃萌）
    - ​**慢眨一下**（温柔/认可/得意）

    ###### 强表情之**耳朵**

    - ​**竖直不动**（认真/警戒/执法）
    - ​**贴后一点**（委屈/护短/黏人）
    - ​**微转向**（困惑/对焦/听诊感）

    ###### 强表情之**尾巴**

    - ​**绷直**（倔强/执法/紧张）
    - ​**绕住**（黏人/占有/安抚）
    - ​**尖尖动一下**（得意/警惕/小情绪）
    - ​**垂着不动**（委屈/失落）
