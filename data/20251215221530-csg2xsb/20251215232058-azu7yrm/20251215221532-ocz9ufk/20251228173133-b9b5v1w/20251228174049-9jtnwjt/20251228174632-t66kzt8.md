# 猫猫型微信号-选部位动作

# 筛选1：原则——【静态动作】+【**微幅度/完全静止】**

###### 静态动作

最后一行是截图式定格画面，那所写的动作，就必须是【一幅图能展现出来的】动作。

包含：

1. 静态动作

    还、仍、一直、就、死死、牢牢、不动、没松

    例：还按着 / 一眨不眨 / 耳朵竖着不动 / 尾巴绷成一条线 / 趴在那儿不挪
2. 图画能表达的动态动作

    很多画都有动感，通过静态截图、展现出动态动感——这类动态动作，也可以写

    水被它推到了嘴边（类比画，就是一只爪爪和一杯水）

###### **微幅度/完全静止**

大动作会像推进，小动作更像截图。

所有优先微动作/微幅度 或 完全静止

# 筛选2：对齐之前的两条

- ​**对齐功能**：这个部位在末行要表达的“唯一信息”是什么？（拦住/守住/确认/委屈/得意…）
- ​**对齐滤镜**：同一信息用什么气质呈现？（一本正经/倔强/困惑/谨慎/委屈/得意）

还有**压字数**：末行通常 **8–14字** 最稳；过长会像叙述，不像截图

# 筛选3：高胜率动作的不断积累

###### 强收束之「**全身姿态 + 位置关系」**

- ​**占位**：趴/蹲/坐在关键位置
- ​**不挪**：不动/不让/堵着
- ​**挡住**：挡在前面/夹在中间

###### 强收束之**爪子细节**

- ​**不松**（状态化：还按着/没松爪）
- ​**压住**（证物/危险物被控制）
- ​**爪尖收着**（谨慎/小心翼翼）
- ​**轻轻搭着**（安抚/确认）

###### 强表情之**头部轮廓**

- ​**歪头**（困惑/确认）
- ​**缩脖**（谨慎/怂但认真）
- ​**抬下巴**（得意/一本正经）
- ​**定住**（加载失败）

###### 强表情之**眼睛**

- ​**一眨不眨**（一本正经/检查/执法）
- ​**圆睁**（震惊/当真）
- ​**半眯**（怀疑/审视/嫌弃萌）
- ​**慢眨一下**（温柔/认可/得意）

###### 强表情之**耳朵**

- ​**竖直不动**（认真/警戒/执法）
- ​**贴后一点**（委屈/护短/黏人）
- ​**微转向**（困惑/对焦/听诊感）

###### 强表情之**尾巴**

- ​**绷直**（倔强/执法/紧张）
- ​**绕住**（黏人/占有/安抚）
- ​**尖尖动一下**（得意/警惕/小情绪）
- ​**垂着不动**（委屈/失落）
