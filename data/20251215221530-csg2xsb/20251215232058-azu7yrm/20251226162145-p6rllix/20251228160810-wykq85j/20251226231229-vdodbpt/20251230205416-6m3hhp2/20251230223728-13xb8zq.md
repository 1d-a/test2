## 2、比作植物或人体

1. 植物：蒲公英绒、花瓣、嫩芽（偏治愈/轻盈）

    梅花小爪
2. 人体/衣饰化的“拟人身体”：小手手、小脚丫、眉毛（萌但语域更口语）

## 3、比作人造物件

**按“用途脚本”分（更互斥）**  ：

1. 软物/纺织物：毛球、绒毯、围脖（治愈软萌）
2. 玩具/文具：橡皮、毛线团、印章（童趣萌）
3. 家居小物：小刷子、小扫帚、小挂件（俏皮萌）
4. 工具/器具：小钩子、小夹子（反差萌：小但“有用/有点厉害”）
5. 科技装置/零件：天线、雷达、探测器（机灵萌/中二萌）
6. 容器/形制：小盒子、团成一团像“抱枕”（场景感强）

    # 根本任务：选核心名词

    本质是：**你在换“命名方式”** ，如头→脑袋

    - 不同的命名方式，画面、语域、亲昵度会截然不同

    ‍

    ‍

    # 一、两种重命名类别：同义切换、微部位化

    同义切换：语域不变

    - 语义和语域都不变，只是更口语/更亲昵/更萌......

    ‍

    微部位化：更具体细小

    - 语域进行改变，更具体、更细小了

    # 二、六常用部位 的 「两种重命名类别+可爱感排序」

    六个常用部位，选自娇小可爱部位的MECE[^2]

    ## 1 **、猫爪**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. **爪团儿**
    2. **猫爪子**
    3. **猫爪**
    4. **爪子 爪爪**
    5. **脚爪**
    6. **脚丫**
    7. ​**前爪**（更具体，但更“描述”而非“卖萌”）
    8. **后爪**
    9. ​**爪**（最硬，最利，最像“武器/爪牙”）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. **肉垫、肉球**（天然萌点）
    2. ​**爪垫**（更中性一点）
    3. ​**趾间毛**（细节可爱，很“镜头”）
    4. **脚趾** / **趾尖**
    5. ​**爪尖**（开始变“锋利感”）
    6. ​**爪甲**（更专业、更冷）

    ‍

    ## 2 **、鼻子**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. ​**猫鼻子**（带一点“你在看它”的亲昵感）
    2. **鼻子/小鼻子**
    3. ​**鼻**（更文气、更硬一点）
    4. ​**鼻部**（偏描述/偏医学口吻）
    5. ​**鼻端**（更专业、最冷）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. ​**鼻头**（很圆很软的联想，常用于萌描写）
    2. ​**鼻尖**（“镜头拉近”，也很常用）
    3. ​**鼻翼**（中性，但很细）
    4. ​**鼻梁**（更偏“形态结构”，可爱感下降）
    5. ​**鼻孔**（细但不太萌）
    6. ​**鼻镜**（很解剖/很专业）

    ## 3 **、耳朵**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. **猫耳朵**
    2. **耳朵**
    3. ​**猫耳**（偏二次元语感一点，但仍可用）
    4. ​**耳**（更简、更硬）
    5. ​**外耳**（更专业）
    6. ​**耳部**（最“说明书”）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. **耳尖/耳朵尖/小耳尖**（萌度最高、最常用的“特写点”）
    2. **耳背** / ​**耳后**（“被挠的地方”，很生活化）
    3. ​**耳缘**（边边一圈，细节感强）
    4. ​**耳根**（更结构化）
    5. ​**耳廓**（解剖味更重）
    6. ​**耳道**（不太萌，偏医学）

    ‍

    ## 4 **、眼睛**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. **大眼睛**（最稳的“萌中性”）
    2. ​**猫眼**（更带物种气质，略俏）
    3. ​**眸子**（更柔美、偏文艺）
    4. ​**眼眸**（更抒情、更漂亮，但不一定“萌”）
    5. ​**眼**（更硬更干净）
    6. ​**目**（最书面/最冷）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. ​**眼角**（非常好用的“情绪萌点位”）
    2. ​**眼尾**（更轻巧、更俏）
    3. ​**眼睫**（有一点“可爱滤镜”，但猫本身不突出，写作可用）
    4. ​**眼睑**（更中性结构）
    5. ​**瞳孔**（更理性、更镜头）
    6. ​**虹膜**（更专业）
    7. ​**瞬膜**（最解剖、最不萌）

    ‍

    ## 5 **、尾巴**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. **猫尾巴**
    2. **尾巴**
    3. **小扫帚/小掸子**
    4. ​**尾**（更硬、更简）
    5. ​**尾部**（更书面/说明书）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. ​**尾尖**（最常用、最萌的“点”）
    2. ​**尾梢**（比“尾尖”更文气一点）
    3. ​**尾毛**（触感/画面更强）
    4. ​**尾根**（更结构化，可爱感下降）
    5. ​**尾椎**（最专业、最冷）

    ‍

    ## 6 **、胡须**

    ### A1 同义命名（更可爱 → 更中性/硬）

    1. **小胡子**（比“胡须”更不“老头感”，更萌）
    2. **胡须**
    3. ​**触须**（更科普/更专业）
    4. ​**须**（更硬更简、偏文言）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. **胡须尖** / ​**须尖**（细而轻，最“萌特写”）
    2. ​**眉须**（有点奇妙的小萌点）
    3. ​**胡须垫**（软肉鼓起来的那块，“可爱但稍专业”）
    4. ​**口角须**（更术语化）
    5. **胡须根** / ​**须根**（最结构化、最不萌）

    ‍

    ## 补：头（猫头 / 脑袋）

    ### A1 同义/同层级命名（更可爱 → 更中性/硬）

    1. ​**猫脑袋**（最口语、最“宠溺视角”）
    2. ​**小脑袋**（虽然带“小”，但它更像“替换成常用昵称核名词”，在小说里非常稳）
    3. **脑袋**
    4. **猫脑袋瓜** / ​**脑袋瓜**（更俏皮，略带口语喜感；“猫脑袋瓜”更萌更明确）
    5. **小脑瓜**（更口语但偏糙一点）
    6. ​**猫头**（更像物种特征称呼，略有俏感）
    7. ​**头**（最中性）
    8. ​**头部**（说明书/镜头描述感，最不萌）

    ### A2 微部位称呼（更可爱 → 更中性/硬）

    1. ​**额头**（很容易写出“乖/委屈/撒娇”的萌情绪位）
    2. ​**脑门**（更口语、更亲昵一点，带点喜感）
    3. ​**额角**（更细、更镜头，适合轻轻一笔）
    4. ​**头顶**（非常常用、画面清晰，萌度中等偏稳）
    5. **发旋** / ​**毛旋**（细节感很强，属“特写萌点”，但不常用：适合你要写得很“观察者”时）
    6. ​**后脑勺**（带点憨萌/搞笑：猫转过去时很可爱）
    7. ​**后脑**（更结构化）
    8. **颅顶** / ​**颅部**（最专业最冷，基本不萌）

    # MECE

    按照在文字中的可爱强度，分为各个等级

    - 文字中的可爱塑造强度，跟视频/画面不太一样

    |可爱级别|部位|
    | --------------| ------|
    |**S 级**（顶级萌点）|**猫爪（肉球）＞小鼻子（含人中）≈耳朵**|
    |**A 级**（高阶萌点）|**大眼睛≈尾巴＞胡须**|
    |**B 级**（特殊萌点）|**肚皮、小尖牙、小舌头、小屁股等**|
