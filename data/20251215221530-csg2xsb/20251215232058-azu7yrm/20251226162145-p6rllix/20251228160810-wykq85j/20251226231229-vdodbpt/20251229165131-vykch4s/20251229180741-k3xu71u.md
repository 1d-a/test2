# 可爱特征排序+对应形容词

# 排序：四大天王+其它

## S级：四大天王→小、圆、软、毛茸茸

这四个最强，也最稳

## A级：

##### 1) ​**比例与钝化：大/短/钝**（“幼态结构”的另一半）

- ​**大**：主要用于眼睛、脑袋（大眼睛、大脑袋）
- ​**短**：用于四肢、尾巴的一截、鼻梁（短短的小腿、小短尾）
- ​**钝/圆钝**：让“尖”变得安全（圆钝的鼻头、钝钝的爪尖）

##### 2 **）胖 / 鼓 / 嘟（“可抱性”）**

##### 3）**粉 / 奶 / 浅色系（颜色的“甜”）**

- 粉、奶白、浅杏色这类色感本身就偏萌
- 例：*粉粉的鼻头 / 奶白的绒毛*

##### 4）**弹 / Q弹（手感升级）**

##### 5）**润 / 亮 / 水光（湿润感、反光感）**

- 适合眼睛、鼻尖、嘴角：会让画面“活”起来
- 例：*水光光的猫瞳 / 湿湿的鼻尖*

##### 6）**温度感（暖 / 热乎 / 凉凉）--主要是暖**

- 暖：治愈、安心；凉：真实、亲近（尤其鼻尖/耳尖）
- 例：*热乎乎的毛肚子 / 凉凉的耳尖*
