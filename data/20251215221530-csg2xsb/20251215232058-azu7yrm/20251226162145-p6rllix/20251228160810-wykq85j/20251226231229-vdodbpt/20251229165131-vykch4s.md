# 感官描绘 adj：

# 一、基本认知

 *“萌态描绘”* ：通过****形容词****刻画部位的尺寸、形状和触感，使之具备典型萌属性。

‍

**本质上就是使用**​ **<u>*——经典可爱特征→“圆”“小”“软”“ 毛茸茸”等*</u>**

对应→****尺寸、形状、质感****

- 尺寸上用“小”“迷你”强调****小巧****
- 形状上用“圆滚滚”“胖乎乎”强调****圆润笨拙****
- 质感上用“软绵绵”“毛茸茸”强调****柔软蓬松****。

‍

> 其实不只是小可爱猫猫，这些词用到任何地方，都会显得可爱
>
> 比如三花娘娘中的庙子很小，小小的神像

## 机制

直接调用“婴儿图式”线索：****小+圆+软****是让生物显幼态的主要信号

‍

## 写法：只写1-2处关键萌点

误用风险主要在于****堆砌形容词****：如果一次性用了过多修饰（如“毛茸茸软乎乎胖嘟嘟的小耳朵”），反而让句子臃肿、矫揉造作。

****技巧****是选1-2个最突出且契合实际的感官印象来描写

> 跟部位选择的写法：只写1-2处关键萌点+情景与排序[^1] 一回事
>
> 所有可爱的，堆多了都有误用风险，都只选一两个

‍

[^1]: ## 写法：只写1-2处关键萌点+情景与排序

    常见误用是****过犹不及****：

    如果通篇只堆砌细碎部位，反而使读者难以拼凑整体形象。

    - 因此通常每次选择1-2处关键萌点细节描写即可，不要把猫解剖成一堆零件来卖萌

    ‍

    #### 选用：情景与排序，两相考虑

    1. ****符合情境需求****
    2. 娇小可爱部位的MECE[^2]

    这两个共同决定了【选写哪个部位】

    ‍


[^2]: # 娇小可爱部位的MECE

    # MECE

    按照在文字中的可爱强度，分为各个等级

    - 文字中的可爱塑造强度，跟视频/画面不太一样

    |可爱级别|部位|
    | --------------| ------|
    |**S 级**（顶级萌点）|**猫爪（肉球）＞小鼻子（含人中）≈耳朵**|
    |**A 级**（高阶萌点）|**大眼睛≈尾巴＞胡须**|
    |**B 级**（特殊萌点）|**肚皮、小尖牙、小舌头、小屁股等**|

    ‍

    # 选用--根据情况选喽

    # 排序：四大天王+其它

    ## S级：四大天王→小、圆、软、毛茸茸

    这四个最强，也最稳

    ## A级：

    ##### 1) ​**比例与钝化：大/短/钝**（“幼态结构”的另一半）

    - ​**大**：主要用于眼睛、脑袋（大眼睛、大脑袋）
    - ​**短**：用于四肢、尾巴的一截、鼻梁（短短的小腿、小短尾）
    - ​**钝/圆钝**：让“尖”变得安全（圆钝的鼻头、钝钝的爪尖）

    ##### 2 **）胖 / 鼓 / 嘟（“可抱性”）**

    ##### 3）**粉 / 奶 / 浅色系（颜色的“甜”）**

    - 粉、奶白、浅杏色这类色感本身就偏萌
    - 例：*粉粉的鼻头 / 奶白的绒毛*

    ##### 4）**弹 / Q弹（手感升级）**

    ##### 5）**润 / 亮 / 水光（湿润感、反光感）**

    - 适合眼睛、鼻尖、嘴角：会让画面“活”起来
    - 例：*水光光的猫瞳 / 湿湿的鼻尖*

    ##### 6）**温度感（暖 / 热乎 / 凉凉）--主要是暖**

    - 暖：治愈、安心；凉：真实、亲近（尤其鼻尖/耳尖）
    - 例：*热乎乎的毛肚子 / 凉凉的耳尖*
